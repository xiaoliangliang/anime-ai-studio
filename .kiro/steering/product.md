# DramaAI - Product Overview

DramaAI (AI短剧工坊) is an AI-powered one-stop short drama creation platform that guides users through the complete workflow from script writing to video generation.

## Core Concept

The platform implements a 5-stage creative pipeline where each stage is assisted by a specialized AI agent:

1. **Screenwriter (编剧)** - Generates story outlines, character profiles, world settings, and episode scripts
2. **Storyboard (分镜师)** - Breaks scripts into shots with camera angles, movements, and timing
3. **Image Designer (图像设计师)** - Creates prompts for character references, scene references, and keyframes
4. **Artist (美工)** - Generates images using AI text-to-image models
5. **Director (导演)** - Generates video clips from keyframes

## Key Features

- Conversational AI interaction for creative guidance
- Infinite canvas (tldraw) for visualizing all stage outputs
- Local-first storage using IndexedDB
- Stale tracking system to mark downstream data when upstream changes
- JSON Schema validation for AI outputs
- i18n support (Chinese/English)

## Target Users

- Short drama creators and screenwriters
- Content creators and self-media producers
- AI video enthusiasts
