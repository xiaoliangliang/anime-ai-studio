## 核心技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端框架** | React 18 + TypeScript + Vite | 现代化前端开发框架 |
| **状态管理** | React Context (ProjectContext) | 轻量级全局状态管理 |
| **画布渲染** | tldraw | 可视化画布编辑器 |
| **样式方案** | Tailwind CSS | 原子化 CSS 框架 |
| **本地存储** | IndexedDB (idb) | 浏览器本地数据库 |
| **后端 API** | Vercel Serverless Functions | 无服务器函数 |
| **AI 文本** | Replicate API (Claude) | 大语言模型文本生成 |
| **AI 图片/视频** | RunComfy (Seedream) | AI 图像和视频生成 |
| **国际化** | i18next | 多语言支持 |
| **部署平台** | Vercel | 全栈部署平台 |

## 数据流说明

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   用户输入   │───▶│  ChatPanel  │───▶│  AI 服务    │───▶│ 结构化 JSON │
└─────────────┘    └─────────────┘    └─────────────┘    └──────┬──────┘
                                                                 │
                                                                 ▼
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│CanvasPanel  │◀───│ IndexedDB   │◀───│ProjectContext│◀───│Schema 校验  │
│  (渲染)     │    │  (持久化)   │    │  (状态)     │    │             │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

1. 用户在 **ChatPanel** 与 AI 对话
2. AI 返回**结构化 JSON**（通过 Schema 校验）
3. 数据存入 **ProjectContext** → **IndexedDB**
4. **CanvasPanel** 根据阶段数据渲染到 **tldraw** 画布

## 环境变量

| 变量名 | 说明 |
|--------|------|
| `REPLICATE_API_TOKEN` | AI 文本生成 API Token |
| `RUNCOMFY_API_TOKEN` | 图片/视频生成 API Token |
| `IMGBB_API_KEY` | 图片上传托管服务 |
| `VITE_ENABLE_IMAGE_GENERATION` | 功能开关：图片生成 |
| `VITE_ENABLE_VIDEO_GENERATION` | 功能开关：视频生成 |
