## 3. 阶段间数据流转

```
+=======================================================================================+
|                                 数据流转示意图                                         |
+=======================================================================================+
|                                                                                       |
|   用户输入                                                                            |
|      |                                                                                |
|      v                                                                                |
|  +--------+        +--------+        +--------+        +--------+        +--------+  |
|  |        |  JSON  |        |  JSON  |        |  JSON  |        |  资源  |        |  |
|  | 阶段1  | -----> | 阶段2  | -----> | 阶段3  | -----> | 阶段4  | -----> | 阶段5  |  |
|  |        |        |        |        |        |        |        |        |        |  |
|  | 编剧   |        | 分镜师 |        | 图像   |        | 美工   |        | 导演   |  |
|  |        |        |        |        | 设计师 |        |        |        |        |  |
|  +--------+        +--------+        +--------+        +--------+        +--------+  |
|      |                 |                 |                 |                 |        |
|      v                 v                 v                 v                 v        |
|  +--------+        +--------+        +--------+        +--------+        +--------+  |
|  | 剧本   |        | 分镜表 |        | 提示词 |        | 图片   |        | 视频   |  |
|  | 数据   |        | 数据   |        | 数据   |        | 文件   |        | 文件   |  |
|  +--------+        +--------+        +--------+        +--------+        +--------+  |
|                                                                                       |
|  数据格式:                                                                            |
|  - 剧本数据: 大纲表格 + 角色设定 + 剧本正文                                           |
|  - 分镜数据: 10字段分镜表 (镜号/地点/画面/景别/机位/运镜/时长/对话/负责人/备注)       |
|  - 提示词: 角色提示词 + 场景提示词 + 关键帧提示词                                     |
|  - 图片: 角色参考图 + 场景参考图 + 关键帧图片                                         |
|  - 视频: 镜头视频片段 + 合成视频                                                      |
|                                                                                       |
+=======================================================================================+
```

## 6. 交互操作说明

```
+=======================================================================================+
|                                   交互操作                                             |
+=======================================================================================+
|                                                                                       |
|  画布操作:                                                                            |
|  +---------------------------------------------------------------------------------+  |
|  |  操作          |  方式                  |  效果                                 |  |
|  +-----------------+------------------------+--------------------------------------+  |
|  |  平移          |  鼠标拖拽 / 触摸滑动    |  移动画布视口                        |  |
|  |  缩放          |  滚轮 / 双指捏合        |  放大/缩小画布                       |  |
|  |  快速跳转      |  点击导航栏阶段按钮     |  跳转到对应阶段位置                  |  |
|  |  编辑内容      |  双击表格/卡片          |  进入编辑模式                        |  |
|  |  选择元素      |  单击                   |  选中元素,显示操作菜单               |  |
|  +-----------------+------------------------+--------------------------------------+  |
|                                                                                       |
|  聊天区操作:                                                                          |
|  +---------------------------------------------------------------------------------+  |
|  |  操作          |  方式                  |  效果                                 |  |
|  +-----------------+------------------------+--------------------------------------+  |
|  |  发送消息      |  Enter / 点击发送      |  发送消息给AI                        |  |
|  |  换行          |  Shift + Enter         |  消息内换行                          |  |
|  |  选择选项      |  点击下拉框/单选       |  设置参数                            |  |
|  |  清空对话      |  点击清空按钮          |  清空当前阶段对话历史                |  |
|  +-----------------+------------------------+--------------------------------------+  |
|                                                                                       |
|  键盘快捷键:                                                                          |
|  +---------------------------------------------------------------------------------+  |
|  |  快捷键        |  功能                                                          |  |
|  +-----------------+--------------------------------------------------------------+  |
|  |  Ctrl + S      |  保存项目                                                      |  |
|  |  Ctrl + Z      |  撤销                                                          |  |
|  |  Ctrl + Y      |  重做                                                          |  |
|  |  1-5           |  切换到阶段1-5                                                 |  |
|  |  Space + 拖拽  |  画布平移                                                      |  |
|  |  + / -         |  缩放画布                                                      |  |
|  +-----------------+--------------------------------------------------------------+  |
|                                                                                       |
+=======================================================================================+
```
