| 功能 | MVP状态 | 说明 |
|------|---------|------|
| 素材状态显示 | ✅ 必做 | 显示分镜数、关键帧数、总时长 |

| 单镜头重新生成 | ✅ 必做 | 对不满意的镜头重新生成 |
| 生成进度显示 | ✅ 必做 | 显示当前生成进度 |
| 时间线编辑器 | ⏸️ MVP后 | 拖拽排序/转场效果 |
| 音频轨 | ⏸️ MVP后 | 配音/音效/BGM |
| 字幕轨 | ⏸️ MVP后 | 自动/手动字幕 |
| 视频合成 | ⏸️ MVP后 | 合成完整视频 |
| 视频导出 | ⏸️ MVP后 | 导出 MP4/MOV |
| 高级生成设置 | ⏸️ MVP后 | API选择/参数调整 |
