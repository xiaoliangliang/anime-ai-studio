# AI短剧一站式工作平台 - 首页与项目管理原型图

> **MVP 说明**：搜索/筛选/排序、项目复制、项目导出功能在 MVP 后实现。

## 1. 首页概述

首页是用户进入平台的第一个页面，提供项目管理和快速创建新项目的功能。参考Lovart的设计风格，采用简洁现代的卡片式布局。

## 9. 交互说明

### 9.1 首页交互流程

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 输入创意并发送 | 在AI输入框输入后点击发送 | 创建新项目并跳转到编剧阶段 |
| 点击类型标签 | 点击"都市爱情"等标签 | 将标签内容填充到输入框 |
| 点击新建项目卡片 | 点击"+"卡片 | 聚焦到AI输入框 |
| 点击项目卡片 | 点击已有项目 | 打开该项目工作区 |
| 悬停项目卡片 | 鼠标悬停 | 显示操作按钮 |
| 点击更多菜单 | 点击"⋮"按钮 | 显示重命名/复制/删除菜单 |

### 9.2 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd + N | 新建项目 (聚焦输入框) |
| Ctrl/Cmd + O | 打开项目列表 |
| Ctrl/Cmd + , | 打开设置 |
| / | 聚焦搜索框 |

## 11. 总结

首页设计要点：
1. **快速创建**: 用户在首页输入创意即可立即创建项目进入编剧阶段
2. **项目管理**: 卡片式展示所有项目，显示项目预览和进度
3. **进度可视**: 每个项目卡片显示5个阶段的完成状态
4. **便捷导航**: 左侧固定导航栏，顶部面包屑导航
5. **类型快选**: 快捷标签帮助用户快速选择剧本类型

### MVP 功能范围

| 功能 | MVP状态 |
|------|---------|
| AI创意输入框 | ✅ 必做 |
| 类型快选标签 | ✅ 必做 |
| 项目卡片网格 | ✅ 必做 |
| 打开/重命名/删除项目 | ✅ 必做 |
| 搜索/筛选/排序 | ⏸️ MVP后 |
| 项目复制 | ⏸️ MVP后 |
| 项目导出 | ⏸️ MVP后 |
| 列表视图模式 | ⏸️ MVP后 |
